# Contribution Guidelines
## Introduction
Contributions are welcome and are greatly appreciated. Every little bit helps, and credit will always be given.
## Getting Started
1. Fork the `incontext-learning` repository on GitHub.
2. Clone the repository and install the dependencies.
## Branching Model
- `main` branch is for stable releases.
- `dev` branch is for ongoing development.
## Commit Messages
- Use the present tense ("Add" instead of "Added").
- Use the imperative mood ("Fix" instead of "Fixes").
## Code Style
Follow PEP 8 for Python code.
